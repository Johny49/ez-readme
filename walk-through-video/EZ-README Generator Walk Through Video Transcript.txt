This video will walk through the steps to create a formatted readme using the easy read me generator tool. 

To start, open terminal and navigate to the project directory. 

Then, type in npm init -y to initialize node package manager. 

After this, type npm install inquirer.  

When inquirer finishes installing, type node index.js. 

You will be presented with a series of questions. Your answers will be used to create the contents of the readme file. 

After answering all the questions you will see a summary of all your answers,  along with a message telling you the readme was created successfully.
